<div>태그는 division의 약자로 웹사이트의 레이아웃(전체적인 틀)을 만들 때 주로 사용

속성		        비고
style    		      스타일
width		     가로 크기
height                      세로 크기
border                     테두리 굵기
background-color        배경 색상
float		         정렬
margin                         여백

<hgroup> 헤딩을 다양화하기 위해 그룹으로 묶어서 <h1>~<h6>요소를 이용함

<nav> 메뉴 또는 링크와 같은 내비게이션을 위해 구성된 섹션을 나타냄

<section> 문서나 애플리케이션의 섹션을 나타내기 위해 추가되었으며, 이 요소는 문서의 구조를
나타내기 위해 h1, h2, h3 등 header에 정의된 요소들과 같이 사용됨

<article> 문서 내의 별도의 글을 표시할 수 있다. 이 요소는 블로그나 뉴스 본문 등을 나타낼 경우 사용함

<aside> 문서의 주 내용이 아닌 관련성이 낮은 내용들을 나타내며, 본문과 직접적으로 상관이 없는 관련
사이트 링크나 광고 등을 나타냄

<footer> 섹션의 꼬리말을 나타내고 저자나 저작권 등을 포함함

반응형 웹 : 반응형 웹은 모바일 환경이 시작되면서 장치도 다양해져 해상도의 크기 및 비율이
제각각 다른 경우에 이를 자유롭게 지원하는 웹 기술이다. 리액트, 뷰, 앵글러 같은 웹 컴포넌트를
많이 활용하여 반응형 웹을 구현한다 -> 디바이스 종류에 따라 웹페이지의 크기가 자동적으로 재조정
